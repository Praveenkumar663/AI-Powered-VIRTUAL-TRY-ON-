# 👕 AI-Powered Virtual Try-On System

## 📌 Overview

This project is an AI-based virtual try-on system that allows users to try clothes digitally in real time using computer vision.

## 🚀 Features

* Real-time virtual try-on
* Uses OpenCV and MediaPipe
* Detects body pose and overlays clothing
* Interactive and user-friendly

## 🛠️ Technologies Used

* Python
* OpenCV
* MediaPipe
* NumPy

## 📂 Project Structure

* `main.py` – Main application
* `Resources/` – Clothing assets
* `requirements.txt` – Dependencies

## ▶️ How to Run

```bash
pip install -r requirements.txt
python main.py
```

## ⚠️ Note

Make sure your webcam is connected.
